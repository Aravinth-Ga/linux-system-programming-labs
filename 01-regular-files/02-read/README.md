advanced ones add loops, buffers, and offset checks.

Simple

- `lab02_read_return_values.c`
- `lab03_read_permission_error.c`

Advanced
- `lab04_read_loop_and_buffer.c`
- `lab05_read_filestat_and__offset.c`
